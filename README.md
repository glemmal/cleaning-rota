# cleaning-rota
Keep track of your shared flat cleaning rota!

Just edit the file main.js with the appropriate jobs and users.
The rota get's updated automatically every week.

```javascript
{
    users: [
      'Temi',
      'Weronica',
      'Reece',
      'Vil',
      'Elias'
    ],
    jobs: [
      'Hallway / Pfand',
      'Shopping',
      'Kitchen',
      'Bathroom',
      'Off'
    ]
  }
```
